조리실 구조

조리실의 구조(Code-List)는 다음과 같습니다.

1. Models

게시판에 사용되어질 모델을 선언합니다.

2. Public

1) css

부트스트랩 사용을 위한 CSS파일

2) fonts

부트스트랩 폰트 사용을 위한 파일

3) img

기본적 이미지 사용을 위한 이미지 파일

4) js

angularJS, 부트스트랩, jquery 및 라우팅 처리에 사용될 js파일 모음

5) views

html파일 모음

3. Server

NodeJS모듈을 선언하고 라우팅 처리를 설정합니다.

4. Server.js

NodeJs를 사용하여 서버를 재구동 시키고 DB연동을 설정 합니다.
